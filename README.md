# Power BI Analytics Portfolio

This repository showcases a collection of Power BI dashboards focused on financial analytics, market insights, and data-driven decision-making.

The purpose of these projects is to demonstrate how raw data can be transformed into structured insights using Power BI, Power Query, and DAX.

---

## Projects

### 1. Portfolio Analytics Dashboard
Analysis of portfolio performance and risk metrics.

Key features:
- Portfolio returns and cumulative performance
- Volatility and risk analysis
- Drawdown tracking
- Asset allocation breakdown

---

### 2. 🏢 Corporate Finance Dashboard
Financial performance analysis of a company.

Key features:
- Revenue growth trends
- EBITDA and profitability margins
- Cost structure analysis
- Profitability insights

---

### 3. Macro Markets Dashboard
Overview of key macroeconomic indicators and market trends.

Key features:
- Inflation trends
- Interest rate movements
- GDP growth analysis
- Market index performance

---

### 4. Data Professional Survey Dashboard
Analysis of survey data to identify trends in the data industry.

Key features:
- Data cleaning and transformation using Power Query
- Salary analysis by job role
- Geographic distribution of respondents
- Programming language preferences
- Work-life balance and satisfaction metrics
